# Data Analytics Project

## Overview

This project demonstrates a complete data analytics workflow, starting from data loading and exploration to dashboard creation and reporting. The objective was to extract meaningful insights from the dataset using Python, MySQL, and Power BI.

---

## Dataset

The dataset contains customer transaction and behavioral data used for analysis and visualization.

### Key Features

* Customer information
* Purchase details
* Product categories
* Discounts and sales metrics
* Customer behavior patterns

---

## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **MySQL** (Data storage and querying)
* **Power BI** (Dashboard creation)
* **Jupyter Notebook**
* **SQL**

---

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python using Pandas.
* Performed initial inspection of rows, columns, and data types.

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends.
* Identified missing values and outliers.
* Generated visualizations to understand customer behavior and sales patterns.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected data inconsistencies.
* Prepared data for further analysis.

### 4. MySQL Database Integration

* Created a MySQL database.
* Imported the cleaned dataset into MySQL.
* Executed SQL queries to answer business-related questions.

### 5. Power BI Dashboard

* Connected Power BI to the processed dataset.
* Built interactive dashboards and visualizations.
* Sales trends, customer insights, and category analysis.

### 6. Reporting

* Summarized key findings and business insights.
* Documented recommendations based on the analysis.

---

## Dashboard Highlights

The Power BI dashboard includes:

* Total Sales Overview
* Customer Purchase Analysis
* Product Category Performance
* Discount Usage Trends
* Interactive Filters and Slicers


---

## Results & Insights

Some of the key insights derived from the analysis include:

* Identification of top-performing products and categories.
* Analysis of customer purchasing behavior.
* Impact of discounts on sales performance.
* Trends and patterns in customer transactions.
* Data-driven recommendations for business improvement.


## Conclusion

This project showcases end-to-end data analytics skills including data cleaning, exploratory data analysis, SQL querying, dashboard development. The insights generated can help support data-driven decision-making and business growth.
